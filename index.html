// About

What is AI Universe?

AI Universe started as a personal project — I kept too many tabs open trying to track what's new in AI, which tools are worth using, and how pricing compares. So I built one place to hold it all.

🤖
Always up to date
A bot runs daily using Claude API and web search to find new tools, update pricing, and track model changes — so the catalog stays current without manual work.
🏆
Tool of the Week
Updated weekly from Product Hunt's top AI launches — no editorial bias, just what the community is actually excited about.
🔍
Find the right tool
Not sure which AI fits your needs? The "Find My AI" quiz asks 3 quick questions and points you to the best match — no fluff.
💸
Free. No ads. No fluff.
Hosted on GitHub Pages. No tracking, no ads, no affiliate links. Just a useful tool for anyone navigating the AI space.
Open Source
The full source code is on GitHub. Feel free to fork it, contribute, or suggest a tool to add.
